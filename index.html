KEY SIGNATURE

There are two signs basically used to write and identify keys of music, these are Flats and sharps

SHARPS (#)

  • key G...# GO
  • key D...## DOWN
  • key A...### AND
  • key E...#### EAT
  • Key B...##### BY
  • Key F...###### FORCE
  • Key C#....####### CAKE

FLATS (b)

  • Key F...b FLAT

  • key B...bb BECOMES

  • key E...bbb EASY

  • key A...bbbb AFTER

  • key D...bbbbb DRINKING

  • key G...bbbbbb GOOD

  • key C...bbbbbbb COKE
